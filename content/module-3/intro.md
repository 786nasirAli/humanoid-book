# Module 3: The AI-Robot Brain (NVIDIA Isaac™)

Welcome to Module 3, where we delve into creating an intelligent brain for your humanoid robot using NVIDIA Isaac™ technologies.

In this module, you will learn:
- Working with NVIDIA Isaac Sim for advanced robotics simulation
- Integrating Isaac ROS packages for perception and navigation
- Implementing bipedal navigation with Nav2
- Generating synthetic training data
- Transferring models from simulation to reality

Let's explore how to create an intelligent control system for your humanoid robot.
