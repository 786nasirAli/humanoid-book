# Welcome to the Humanoid AI Book

This comprehensive guide teaches you how to build and control humanoid robots using ROS 2 (Robot Operating System 2).

## Course Overview

This course is divided into four comprehensive modules:

### Module 1: The Robotic Nervous System
Learn the fundamentals of ROS 2 architecture, including nodes, topics, services, and how to create an effective communication system for your humanoid robot.

### Module 2: The Digital Twin (Gazebo & Unity)
Explore how to create accurate digital twins of your robot using Gazebo for physics simulation and Unity for advanced visualization.

### Module 4: Vision-Language-Action (VLA)
